INSTRUCTIONS POUR CREER LE FICHIER .env
========================================

1. Créez un fichier nommé exactement ".env" (avec le point au début)
   à la racine du projet (même niveau que package.json)

2. Copiez ce contenu dans le fichier .env :

DATABASE_URL="postgresql://postgres:VOTRE_MOT_DE_PASSE@db.xxxxx.supabase.co:5432/postgres"
NEXTAUTH_URL="http://localhost:3000"
NEXTAUTH_SECRET="uRSsIgP+KdxybZxaPuPDowgcKWLrmdj7QhOMNfjzGLQ="
NODE_ENV="development"

3. Remplacez DATABASE_URL par votre vraie connection string
   (voir GUIDE_DATABASE.md pour les instructions détaillées)

IMPORTANT : 
- Le fichier doit s'appeler exactement ".env" (pas ".env.txt" ou autre)
- Remplacez VOTRE_MOT_DE_PASSE par le mot de passe de votre base de données
- Ne partagez JAMAIS ce fichier !
